# Usage Guide

Using BountySleuth is simple:

1. **Open Dashboard**: Click on the 🔍 BountySleuth icon in your browser extension toolbar to open the analyzer dashboard.
2. **Review Badge**: The extension badge will display the total number of findings.
   - 🔴 **Red**: High severity / Large count of findings.
   - 🟠 **Orange**: Medium / Low severity findings.
3. **Passive Scanning**: Simply browse your target application as a normal user would. BountySleuth will passively analyze headers, scripts, and background requests quietly.
4. **Active Interaction**: Interact with the application heavily. Fill out forms, click buttons, let API requests run. This will populate the **Live API Collector** and trigger **XSS** / **CSRF** module analysis.
5. **Exporting Tools**: Need to write your bug bounty report? Export your findings seamlessly using the **📋 MD** (Markdown) or **💾 JSON** buttons available inside the extension popup. This will give you instant vulnerability details directly for your reports.
